# 《论中国》
## 第一章·中国的独特性
### 实然拆解分析 · 基于王老师方法论 v2.4

## 一、动笔前三件事

### 第一件：根规则确认

**根规则：中国在文明形成期从未遭遇可与之并立的外部文明实体，使其对外关系制度天然以等级性秩序为底层硬约束，而非以主权平等秩序为底层硬约束。**

**生成性检验**：第一章所有现象——朝贡体系为何是等级性而非平等性的、"以夷制夷"治边策略为何以离间管理而非直接征服为目标、围棋战略（战略包围/积累态势）为何优先于国际象棋战略（正面决战）、孙子兵法为何以心理-政治优势先行而非以军事消耗为优先、郑和下西洋为何在技术峰值时主动退出、异族征服者（蒙古/女真）为何最终被汉化而非反向同化——全部可从"从未遭遇可与之并立的外部文明实体"这一根规则自然推导出来，不需要额外假设。✓

**不可更改性检验**：这是物质性锁定，由地缘物理约束产生——喜马拉雅山脉、戈壁沙漠、西伯利亚冻原、太平洋——从汉代至清代，任何单一主体都无法单方面改变地缘物理屏障封堵外部对等文明实体进入的结构性事实。✓

**换人不换结论检验**：汉武帝、唐太宗、明成祖、乾隆帝——换掉所有具体帝王，只要地缘物理约束不变、外部对等文明实体不出现，对外关系的底层逻辑必然维持等级性秩序而非主权平等秩序。朝贡体系的具体操作者可以更换，但朝贡逻辑不会改变。✓

---

**根规则诊断（排除捆绑规则的三种失格）**：

**排除"两个维度捆绑"**：可能的误写版本——"地缘隔绝×文明生产力代差的剪刀差"。检验：文明生产力代差可以从"从未遭遇外部对等文明实体"直接推导出来——没有外部对等文明的比较刺激，中国的技术革新动力不强，这是根规则的派生结果，不是并列规则。真正的单条根规则是"从未遭遇外部对等文明实体"。✓

**排除"机制+后果捆绑"**：可能的误写版本——"中国是区域内唯一文明中心，因而其对外关系必然以等级性秩序运行"。检验：后半句（等级性秩序）是推导结果，不是制度结构本身。根规则只陈述机制：从未遭遇可与之并立的外部文明实体。等级性秩序留给推导链条展示。✓

---

### 第二件：分析终点确认

**主形态：形态一（制度均衡）**

朝贡体系作为中国对外关系的制度形态，在两千年内是高度稳定的自我维持均衡，而非偶然性的政策选择。其稳定条件由物质性约束直接生产，不依赖任何主体的主观意志。

**叠加形态：形态三（约束体系比较）**

围棋逻辑/孙子兵法 vs. 国际象棋逻辑/克劳塞维茨：这是同一章中基辛格明确提出的制度比较结构。两套战略约束体系的根规则差异，以及由此产生的不同战略结果，是形态三的核心分析对象。

**形态叠加方式**：形态一处理"朝贡等级秩序为何稳定"；形态三处理"中国与西方战略约束体系的决定性差异在哪里"。两者不冲突，是同一根规则在不同分析维度的表现。

*识别依据*：第一章没有呈现"设计功能与运行功能背离"的自我对抗（形态二不适用）；两套战略约束体系的对比是基辛格的核心论点之一，且两者的差异来自各自面对的外部威胁性质的不同，而非孰优孰劣。

### 第三件：实证锚点

**锚点一（必然性）：郑和下西洋后主动退出（1433年）**

1405-1433年，明朝郑和率舰队七下西洋，舰队规模远超同期葡萄牙探险舰队。1433年后，明朝主动终止航海计划，官方销毁航海记录。

这证明的不是"中国没有航海能力"，而是：在等级性朝贡秩序的制度逻辑下，海洋扩张没有可以接入的制度目的——因为海外没有可以纳入朝贡秩序的文明主体，扩张没有制度意义。

**必然性检验**：换掉明成祖、换掉郑和，在同样的制度结构（等级性秩序无法被海外扩张所强化）下，任何主政者面临儒家官僚集团对航海政策的系统性反对（宦官主导的航海扩张与儒家士大夫的制度利益直接冲突），都会得出同样的结论：停止没有制度框架支撑的海洋扩张。这是制度结构的必然，不是具体人物的主观选择。✓

**锚点二（必然性）：征服者汉化——蒙古、女真两次重复**

蒙古人于1279年征服中国，建立元朝；女真人于1644年征服中国，建立清朝。两次征服，结果相同：征服者最终都以中华官僚制度和文化框架运作统治，征服者的原始文化逐渐被纳入中华文明轨道，其发源地成为中国的一部分。

这证明的是：只要中华制度框架是区域内唯一的高效统治技术，任何征服者要维持稳定统治就必须进入这套框架——这是结构性必然。换任何征服者民族（蒙古、女真、或假想的其他游牧民族），只要没有替代性的等效文明框架存在，汉化都是唯一可行的统治路径。✓

**锚点三（必然性）：1962年毛泽东援引唐朝战役先例**

基辛格在前言中记录：毛泽东在1962年中印边境战争前召集高级领导人，援引1300年前唐朝中印战争的历史先例制定战略方针，随后发动突然袭击后主动撤回战前控制线，归还印军武器。

这证明的是：同样的制度性问题（边境管理、在等级性秩序中的威慑信号发送）在1300年后产生了同样的制度性解法。换掉毛泽东，任何面对相同制度性问题的领导者，在同样的历史制度框架中都会找到相同的先例。这是制度记忆的稳定性，不是毛泽东的个人偏好。✓

## 二、实然拆解

### 第一步：剥离情绪/道德外衣，还原客观事实

以下是第一章的核心客观行为事实，不含价值标签：

**关于中国的对外关系制度**：从秦统一（前221年）到1842年鸦片战争，约两千年内，中国从未在主权平等基础上与外部政治体建立持续的双边关系。中国建立礼部（管理藩属国）和理藩院（管理游牧部落）处理对外关系，直至1861年才在西方压力下建立类西方外交部机构（总理各国事务衙门），且清廷官员将其定位为"临时应急机构"。

**关于治边策略**：中国官员明确记载的治边方针包括：以通商为诱饵"羁縻"边疆力量；"以夷制夷"——利用外部势力之间的矛盾使其互相牵制；"五饵"策略（汉代贾谊）——通过物质赏赐瓦解游牧部落的战斗意志。目标是"离间驯服周边国家，而不是直接占领"。

**关于战略文化**：围棋（横竖各19条线，通过积累占领地盘最终以微弱优势取胜）与国际象棋（通过消耗或一着妙手将对方国王逼入绝境）构成两种截然不同的战略逻辑。《孙子兵法》的核心论点："上兵伐谋，其次伐交，其次伐兵，其下攻城"——以心理和政治手段取胜先于军事手段。

**关于航海史实**：郑和舰队1405-1433年远航，其目的不是征服或贸易，而是"宣示皇帝德威、收取朝贡"。1433年停止后，航海记录被销毁。中国从未在海外建立殖民地。

**关于异族征服**：蒙古建元朝（1271-1368），女真建清朝（1644-1912），两个朝代最终都以儒家官僚制度运作，皇帝以汉语为官方语言、以中华礼仪为政治正当性来源。

---

### 第二步：锚定支撑事实的制度硬结构

上述所有事实的底层制度硬结构有三个相互支撑的物质性支柱：

**支柱一：地缘物理屏障封堵外部对等文明实体的进入**

喜马拉雅山脉（南）、戈壁-西伯利亚（北-西北）、太平洋（东）构成了三个方向的物理屏障。在工业革命前的技术条件下，这些屏障的隔绝效果是绝对性的：任何与中国文明发展水平相当的外部文明（古罗马、伊斯兰世界、印度文明）都无法对中国形成持续的军事-政治压力。周期性出现的外部威胁，全部来自北方游牧部落——技术和组织水平系统性低于中国农耕文明的边缘力量。

**支柱二：文明生产力代差的单向性**

在前工业时代，中国农耕文明的生产力（粮食产量、人口规模、手工业技术、官僚组织效率）系统性高于其所有邻居。直至1820年，中国GDP占世界总量30%以上，超过西欧、东欧与美国之和。这种代差是生态性的，不是政策性的：南方的稻作农业和北方的旱作农业构成了区域内最高效的粮食生产系统，这不是任何政治决策能够复制的技术基础。

**支柱三：儒家官僚制度产生了以维护等级性秩序为职业利益的阶层**

儒家经典作为科举入仕的首要条件，产生了一个知识垄断集团，其职业合法性完全依附于等级性秩序的维持。这是制度性锁定叠加在物质性锁定之上：即使地缘屏障被突破（例如北方游牧民族的南下），儒家官僚体系的惯性仍然是等级性秩序的制度维持者——征服者需要这套官僚机器来统治，因此不得不接受这套机器所承载的等级性秩序逻辑。

---

### 第三步：分析制度结构联动逻辑，推导行为的必然性

从根规则（从未遭遇外部对等文明实体）向下推导：

**推导链条A——对外关系制度：朝贡体系为何是等级性秩序的必然产物**

根规则：从未遭遇外部对等文明实体
→ 中国与外部政治体的接触，始终是文明技术代差巨大的双方之间的接触
→ 外部政治体对中国市场、技术、农业知识有单向需求
→ 中国对外部政治体的政治承认有单向给予能力
→ "平等"在这个结构中没有制度意义——两个资源禀赋严重不对等的主体，没有建立平等框架的现实基础
→ 因此，朝贡等级秩序必然产生：中国以承认其政治合法性（册封）和允许贸易（通商）为条件，外部政治体以表达臣服（朝贡礼仪）为回报
→ 这不是中国统治者的傲慢，而是代差结构的制度化表达

**推导链条B——治边策略：离间管理为何优先于直接征服**

根规则 → 外部威胁来自游牧边缘力量，技术-生产力低于中国
→ 游牧力量的优势在于机动性，劣势在于分散性和对农耕商品的依赖
→ 直接征服游牧地带的成本极高（草原不产粮食，无法用农耕管理方式统治），收益极低（游牧地带无法贡献稳定税收）
→ "以夷制夷"——利用游牧部落间的内部矛盾使其相互消耗——成本最低
→ "五饵"——通过赏赐瓦解游牧部落战斗意志——次优选择
→ 离间管理的目标不是消灭边缘威胁，而是维持"边缘威胁不联合"的均衡状态
→ 这正是孙子"不战而屈人之兵"在治边层面的制度化表达

**推导链条C——战略文化：围棋逻辑为何优先于国际象棋逻辑**

根规则 → 外部威胁不是可以决战消灭的对等文明实体，而是流动性强的边缘游牧力量
→ 决战（国际象棋逻辑）对游牧力量无效：游牧军队被击败后可以撤退，无法被彻底消灭；且中国军事中心在南方，游牧威胁在北方，长距离正面决战的后勤成本极高
→ 战略包围（围棋逻辑）更有效：通过构建联盟、离间敌对部落、控制关键地理节点，使游牧力量无法形成统一的战略合力
→ 孙子的"伐谋"（分解敌方战略意志）和"伐交"（瓦解敌方联盟）先于"伐兵"（正面战争），是这一战略结构的理论提炼
→ 围棋战略和孙子兵法不是中国统治者的主观偏好，是面对边缘游牧威胁这一制度性现实时的结构性最优解

**推导链条D——郑和放弃：最强航海力量主动退出的制度性逻辑**

根规则 → 对外关系制度以等级性朝贡秩序为唯一框架
→ 海洋扩张的制度意义只能在朝贡秩序内被定义：宣示皇帝德威、收取朝贡
→ 郑和舰队的目的确实是"宣示皇帝德威，而非征服"——这本身就是朝贡秩序的制度逻辑约束
→ 海洋扩张无法产生稳定的朝贡关系（远洋目的地不具备被纳入东亚朝贡体系的地理邻近性）
→ 航海扩张的边际收益（象征性朝贡）低于边际成本（宦官主导的庞大舰队开支）
→ 儒家官僚集团以等级性秩序为制度利益基础，系统性反对宦官通过航海扩张获取制度外的权力和财富
→ 停止航海是朝贡制度逻辑与官僚利益结构联合作用的必然结果，不是短视政策失误

**推导链条E——征服者汉化：结构必然而非文化优越**

根规则 → 区域内不存在与中华文明并列的对等统治技术
→ 任何征服者（游牧出身）统治中国都面临同一问题：如何管理人口数千万的农耕定居文明
→ 游牧统治技术（部落军事组织）不能直接管理定居农耕经济——征税、司法、工程建设都需要专业化的科层官僚体系
→ 中华儒家官僚体系是区域内唯一成熟的大规模定居文明管理技术
→ 征服者必须接入这套技术才能维持稳定统治，而这套技术以中华文明的等级性价值观和语言为载体
→ 征服者一旦接入，其文化被技术载体反向同化是结构性必然

### 第四步（形态一）：均衡的稳定条件与崩溃条件

**均衡形态**：等级性朝贡秩序在地缘物理屏障完整、文明生产力代差维持的条件下，是高度稳定的自我维持均衡。

**稳定条件**：
- 周边外部威胁的技术-生产力水平系统性低于中国，任何试图以平等框架与中国互动的外部政治体，都在贸易和军事两个维度上处于谈判劣势，主动接入朝贡秩序的成本低于拒绝接入
- 地缘屏障维持了"外部对等文明实体不出现"的物质条件，使朝贡秩序的等级性假设从未被根本挑战

**自我维持机制**：
等级性秩序稳定 → 周边政治体通过朝贡获取贸易利益 → 朝贡参与国数量增加 → 朝贡网络的覆盖范围扩大 → 等级性秩序的物质基础（贸易网络）进一步强化 → 均衡自我强化

**崩溃条件**：
当外部出现满足以下条件的政治体时，均衡被打破：
1. 技术-生产力水平与中国对等或超越（工业文明技术）
2. 进入路径绕过地缘屏障（海路进入，而非陆路边境压力）
3. 不需要通过朝贡秩序获取资源（自身即为资源技术的输出方）

满足以上三个条件的政治体：19世纪的英国。鸦片战争（1842年）是均衡崩溃的制度触发点——不是因为英国"更强大"，而是因为英国是第一个彻底绕过了朝贡秩序制度假设的外部力量：它不需要中国承认其合法性，它用军事技术优势直接要求对等条约关系。

## 三、跨域联动

同一根规则（从未遭遇外部对等文明实体）在三个领域的并行运作，且三者相互强化：

**领域一：外交制度领域**——等级性朝贡秩序（礼部、理藩院、朝贡礼仪）

**领域二：战略文化领域**——围棋逻辑/孙子兵法（战略包围、心理优先、"势"的积累）

**领域三：文化制度领域**——征服者汉化机制（中华官僚技术的唯一性驱动同化）

**联动机制**：

外交等级性秩序（领域一）→ 要求战略文化与之匹配：对外威胁不是对等文明竞争者，而是需要管理的边缘力量 → 驱动战略文化发展为围棋式/孙子式的管理逻辑（领域二）

战略文化的管理逻辑（领域二）→ 边缘威胁被管理而非消灭 → 游牧力量周期性入侵时被纳入中华统治技术体系 → 文化同化机制被反复触发（领域三）

文化同化机制（领域三）→ 每次征服都以征服者接入等级性秩序告终 → 朝贡体系的制度假设从未被征服推翻 → 外交等级性秩序的路径依赖进一步加固（领域一）

**联动检验**：去掉任何一个跨域连接，结论都不完整。

- 仅看朝贡体系：无法解释郑和放弃——需要战略文化的"对外扩张无制度意义"来说明
- 仅看战略文化：无法解释征服者汉化——需要"中华统治技术的区域唯一性"来说明
- 仅看文化同化：无法解释朝贡体系的等级性假设为何从未被根本挑战——需要"从未遭遇对等文明实体"的根规则来说明

## 四、历史制度演化

### 第一步：识别关键节点

**节点一：秦统一（前221年）**

选择了什么：以中央集权官僚制度（郡县制）统一政治中国，取代封建诸侯多中心体系

当时存在的其他选项：延续战国格局——多中心城邦体系（欧洲后来实际走向的路径）。春秋战国时期，中国事实上已经发展出了类似欧洲多国体系的制度形态，诸侯之间有外交谈判、联盟体系、战争与和平。

为什么选了这个而非其他（制度性原因）：北方游牧民族（匈奴）的周期性军事压力，要求有能力统一调配农耕文明资源的中央集权体系。分散的诸侯体系无法协调跨区域的长城修建和骑兵组建。中央集权不是皇帝的权力欲望，是应对游牧军事压力的结构性选择。

反事实检验：如果秦选择延续多中心体系，北方游牧压力会驱动其中某个诸侯国通过联合其他诸侯的方式逐渐实现中央集权——路径可能更慢，但中央集权化的压力不变。这个节点是真正的选择分叉点。

**节点二：汉代朝贡体系制度化（前2世纪）**

选择了什么：将对外关系制度化为等级性朝贡秩序，而非延续战国时代诸侯之间的对等外交模式

当时存在的其他选项：汉初确实存在对等框架的尝试——汉高祖与匈奴的"和亲"协议，事实上承认了双方的对等性（甚至暗含匈奴优势）。汉武帝之前，汉朝是以实质上的对等姿态与匈奴共存的。

为什么选了汉武帝的朝贡强化而非和亲延续：汉武帝时期（前141-前87年）农耕文明的生产力（人口、铁器农具、税收）已经积累到可以在军事上对匈奴形成压制优势，和亲的物质基础（不得不进行的妥协）消失了。生产力代差一旦扩大到可以军事反制，等级性秩序的制度化就具备了物质基础。

**节点三：1433年郑和航海终止**

选择了什么：在拥有世界最强航海技术的条件下，主动退出海洋扩张路径，销毁航海记录

当时存在的其他选项：以郑和的航海技术和宝船规模，明朝理论上可以在海外建立贸易站或殖民地（同期葡萄牙正是这么做的）

节点的溯源深度检验：当时真的存在真实的替代选项吗？存在，但有约束——海外扩张路径要求宦官集团持续主导朝廷的对外政策，而这在儒家官僚体系主导的政治结构中不具备制度可持续性。这个节点的真正选择分叉点，是朝廷是否能形成支持海洋扩张的持续制度安排，而非仅靠皇帝个人意志。永乐皇帝去世后，这一制度安排立即崩溃，说明它从未成为稳定的制度框架。

---

### 第二步：识别路径依赖的形成机制

**物质性锁定（主锁定机制）**：

地缘物理屏障是核心锁定机制。这不依赖任何利益集团维持——喜马拉雅山脉和戈壁沙漠不需要儒家官僚来维持其隔绝功能。物质性锁定封堵的是"外部对等文明实体出现"这一根规则失效的可能性。只要工业革命不发生（技术条件），或海路突破地缘屏障的能力不出现（航海技术），物质性锁定就不会松动。

物质性锁定的含义：任何政治改革都无法改变这一锁定。即使某个皇帝决定以"主权平等"为对外关系框架，其周边的游牧部落也不具备接受这一框架的制度能力（他们没有形成可以签署对等条约的主权国家机构）。

**制度性锁定（次锁定机制，叠加在物质性锁定之上）**：

儒家官僚体系是制度性锁定的利益集团：
- 科举制度以儒家经典为核心，官员的职业合法性来自对等级性价值观的维护
- 郑和停止航海的直接推手正是儒家官僚的系统性反对——航海扩张产生了等级性秩序框架外的财富和权力，威胁到士大夫阶层对制度性资源的垄断
- 每次改革试图引入对等性外交框架（如清代某些官员的洋务主张），都必须通过这套官僚机器执行，而官僚机器的运作逻辑是等级性秩序

---

### 第三步：展示路径累积的正反馈

**路径加固的完整链条**：

秦统一（前221年）确立中央集权框架
→ 汉武帝朝贡体系制度化（前2世纪）：等级性秩序获得制度形式
→ 魏晋南北朝（220-589年）游牧民族南下：征服者汉化的第一次大规模重复（五胡十六国最终都进入汉化轨道）→ 路径依赖第一次被重大压力测试并通过
→ 隋唐（581-907年）重统一：等级性秩序在更大规模上被重建，朝贡体系辐射至朝鲜半岛、日本、越南
→ 宋朝（960-1279年）军事劣势期：无法以军事维持等级性优势时，转向以经济优势维持朝贡关系（以大量商品赏赐换取边境和平）——等级性秩序的灵活变体，而非秩序转型
→ 元朝（1271-1368年）征服者汉化：最大规模的外部冲击，最终以征服者进入中华制度框架告终 → 路径依赖第二次被重大压力测试并通过
→ 明朝（1368-1644年）郑和放弃：等级性秩序的制度逻辑主动抵制了可能引入新秩序框架的海洋扩张路径
→ 清朝（1644-1912年）征服者汉化：第三次重大压力测试，结果仍然相同
→ 到19世纪：朝贡等级秩序经历了2000年反复验证的路径累积，制度惯性强度达到最高点

每一个节点都在前一节点的路径约束下做出了顺路选择，且每次对路径依赖的重大压力测试（外族入侵）都以路径的自我恢复告终，进一步加固了"等级性秩序是唯一可行框架"的制度信念。

---

### 第四步：评估19世纪初的路径锁定强度

**受益利益集团的规模**：

到19世纪初，受益于等级性秩序的主体包括：
- 整个儒家士大夫阶层（通过科举进入官僚体系的几十万人）
- 依托朝贡贸易体系获益的沿海商人集团
- 皇权本身（等级性秩序是皇帝政治合法性的制度基础）

**路径自我封堵机制**：

任何试图引入主权平等框架的改革，都必须通过儒家官僚体系执行。但这套体系的逻辑是等级性的，它只有一套处理对外关系的制度语言（朝贡、封赏、羁縻），没有处理"对等主权国家"的制度接口。英国使团1793年到访，清廷坚持要求英国使节磕头（叩头礼），这不是清廷傲慢，而是其制度框架内没有"不需要表达臣服的外来访客"这一概念分类。

**外部条件的约束**：

到1840年，工业革命完成了对地缘物理屏障的技术性突破——蒸汽动力军舰可以进入中国沿海，地缘屏障的物质性锁定开始松动。这是外生冲击，不是内生矛盾积累的结果。

---

### 第五步：连接到当前分析

如果没有这个历史路径，中国在19世纪遭遇西方时的全部反应就无法理解：

清廷把英国定位为可以"以夷制夷"的蛮夷之一（传统治边逻辑），试图用其他欧洲国家制衡英国，这不是无知，而是两千年制度经验的自然应用。清廷在鸦片战争初期拒绝"对等条约"，不是僵化，而是"对等"这个概念在其制度框架内没有位置——从未遭遇过需要被平等对待的外部政治体。

正是这个历史路径，解释了根规则为何在19世纪仍然具有如此强的不可更改性：不是清朝统治者没有能力学习，而是两千年路径积累产生的制度惯性重量，远远超过任何单一外部冲击在短期内能够撬动的范围。

## 五、结构推论

### 形态一：制度均衡结论

在"区域内不存在外部对等文明实体"的物质性约束下，等级性朝贡秩序是高度稳定的制度均衡，而非中国统治者主观选择的外交政策。

**稳定条件**：任何周边政治体试图以平等框架与中国互动，都在贸易和军事两个维度上处于谈判劣势，因此主动接入朝贡秩序的成本低于拒绝接入。没有任何主体有动机推动平等框架的形成，因为代差结构决定了"平等"对较弱方反而不利。

**自我维持机制**：朝贡接入 → 贸易利益 → 更多政治体接入 → 朝贡网络扩大 → 等级性秩序的合法性基础强化 → 均衡进一步稳固。

**崩溃条件**：当满足三个条件的外部政治体出现时——技术生产力对等、进入路径绕过地缘屏障、不依赖朝贡秩序获取资源——均衡的物质基础瓦解。19世纪英国是触发条件的首个具体实例。均衡崩溃不是因为清朝官员的失误，而是物质条件改变后均衡的必然重组。

---

### 形态三：约束体系比较结论

中国战略约束体系（围棋逻辑/孙子兵法）的根规则是：**外部威胁是技术-生产力劣于自身的边缘流动性力量，决战无法彻底消灭威胁，战略包围和心理威慑是最优解**。在这一约束下，"势"的积累（战略态势的逐渐改善）、不战而屈人之兵、迂回包围而非正面突破，是结构性最优策略。

西方战略约束体系（国际象棋逻辑/克劳塞维茨）的根规则是：**外部威胁是技术-生产力对等的多极竞争主体，战略目标是在对等竞争中通过消耗或关键节点突破形成优势**。在这一约束下，兵力集中于"重心"（Schwerpunkt）、寻求决战、歼灭对手主力，是结构性最优策略。

**决定性差异**：不在于哪套体系"更聪明"或"更先进"，而在于**外部威胁的相对技术-生产力水平**这一变量上的不同。

中国战略体系在**外部威胁技术-生产力低于自身**的条件下有效，在外部威胁技术-生产力对等或超越时失效。西方战略体系在**多极对等竞争**的条件下有效，在单极主导的圈层式等级结构下无法解释行为逻辑。

两者不是优劣关系，是适用边界不同。当外部威胁的性质发生根本改变（从游牧边缘力量到工业帝国），中国战略约束体系的适用边界条件消失，战略失效成为结构性必然，而非具体决策者的失误。
